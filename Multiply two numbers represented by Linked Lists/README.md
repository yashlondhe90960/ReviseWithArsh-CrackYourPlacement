# Multiply two numbers represented by Linked Lists

## Description

Given two numbers represented by linked lists, write a function that returns the multiplication of these two linked lists.
Example 1:


```
Input : 9->4->6
        8->4
Output : 79464

Input : 3->2->1
        1->2
Output : 3852
```
